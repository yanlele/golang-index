## 语言结构容器


### 数组

**申明**                              
方式1： `var 数组变量名 [元素数量]Type`                                                 
方式2： `var q [3]int = [3]int{1, 2, 3}`                                   
方式3： `q := [...]int{1, 2, 3}`                           


**遍历数组**                                    
`for index, value := range arr {}`         


**举例子**                             
```go
package main

import "fmt"

func main() {
	a := [...]int{1, 2, 3}
	fmt.Println(a[0])
	fmt.Println(a[len(a)-1])

	for index, value := range a {
		fmt.Printf("index: %d, value: %d \n", index, value)
	}
}
```
